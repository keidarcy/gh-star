# `gh start` GitHub CLI extension

[GitHub CLI](https://github.com/cli/cli) extension for quickly fuzzy finding starred repo using [fzf](https://github.com/junegunn/fzf#readme).

## Installation
```
gh extension install keidarcy/gh-star
```

## Usage
```
gh star
```
